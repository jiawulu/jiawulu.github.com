## 基本使用

1. cmd + D 
2. cmd + shift + D
3. cmd + T
4. cmd + W
5. cmd + Q

## 编辑

1.   选中即复制
2. cmd + F  ; TAb ; cmd + V
3. cmd + shift + H : history
4. cmd + option + B : 快照回放
5. cmd + option + E : tab 选择
